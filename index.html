📧 Lead Capture Setup (Save Emails to Your Own Google Sheet)
⚠️ Important: Without this setup, any Full Name / Email / Mobile Number submitted on your sales page is NOT saved anywhere — it just disappears. Follow the steps below (takes about 2 minutes) to actually capture and save leads.
Step-by-Step: Google Sheet + Apps Script (Free, No Server Needed)
  1. Go to sheets.google.com → create a new blank Sheet.
  2. In that Sheet, click Extensions → Apps Script.
  3. Delete any starter code, then paste the code below into the editor.
  4. Click Deploy → New deployment → gear icon → select Web app.
  5. Set Execute as: Me and Who has access: Anyone, then click Deploy (authorize when prompted).
  6. Copy the Web app URL it gives you, and paste it into the field below.
